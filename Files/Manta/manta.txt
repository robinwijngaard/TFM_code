Installation

wget https://github.com/Illumina/manta/releases/download/v${MANTA_VERSION}/manta-${MANTA_VERSION}.release_src.tar.bz2
tar -xjf manta-1.6.0.release_src.tar.bz2

mkdir build && cd build

../manta-1.6.0.release_src/configure --jobs=4 --prefix=./
make -j4 install


manta s'ha de correr amb python2! fer environment

conda create -n mantaenv python=2.7

Configuration

change in advanced conf options: (in configManta.py.ini file)

minEdgeObservations = 2
minCandidateSpanningCount = 2

the default is 3.


Per els bams: s'han d'especificar un per un, per la qual hem creat un array amb els bams duna carpeta que recullim en un .txt en la carpeta results manta. El problema està que s'han d'espeficiar un per un en el bash.

LA primera part es crear el fitxer de configuració mitjançant el script runmanta.sh. que requereix un argument ($1) que será el nom del projecte.

Additional options informations:

python2 configManta.py --allHelp


El bedfile ha de ser: bgzip-compressed and tabix-indexed
cat ICR96_hg38_noSNP.bed | bgzip > ICR96_hg38_noSNP.bed.bgz

En batch grans, error: too many open files. HE introduit: ulimit -n 75000
